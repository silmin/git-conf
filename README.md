# git-conf
auto git setting

## settings

- Set `user.name` and `user.email` from input
- Set `color.ui` to `auto`
- Set `core.editor` to `vim`

Set these automatically.

## usage
type
```
   $ git clone [this repo]
   $ cd [this repo]
   $ ./autoset.sh
```
and input your username and email.

done.
